# Unlock everything — Free vs Pro

**Trivia Dash** is fully playable for free. **Unlock everything** (Pro subscription) removes limits and opens premium modes.

Prices: **£3.49/mo** · **£19.99/yr** · Product IDs: `monthly`, `yearly`

---

## Free (everyone)

| Feature | What you get |
|---------|----------------|
| **Quick Match (solo)** | Single-player runs vs the clock |
| **Daily challenge** | Same 10 questions for everyone each day |
| **Daily leaderboard** | See today’s top scores (submit your score) |
| **Category practice** | **3 practice runs per day** — pick a category, grind toward 50 correct |
| **Wedge profile** | Tap the category wheel → progress bars per category |
| **Weekly themed event** | Play during the theme week (e.g. Science week) |
| **Pass & Play / Party join** | Couch play; join friend rooms with a code |
| **Trophy Case** | Earn achievements, frames & badges from play |
| **Bot difficulty** | **Easy** bots only in solo/ranked-style bot matches |
| **Share card** | Image + text share for daily results & wedge unlocks |
| **Community packs** | 18 free packs |
| **AI voices** | Steve Harvey + 2 free hosts (neural cloud TTS when Supabase TTS is deployed) |
| **Push-style reminder** | In-app daily nudge (enable in settings) |
| **Question archive** | Free question pool |
| **Ads** | Banner + occasional interstitial/rewarded offers |
